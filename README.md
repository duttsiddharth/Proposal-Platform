# Proposal Automation Platform — SD Advisory Bid Desk

A zero-cost, browser-only proposal generator. Six specialist agents assemble a full
proposal pack (solution architecture, cost estimation, project plan, resource plan,
risk register, implementation roadmap) from a built-in delivery knowledge base.

No API keys, no backend, no build step — a single `index.html`.

## Deploy on GitHub Pages
1. Create a new repo (e.g. `proposal-platform`)
2. Upload `index.html`
3. Settings → Pages → Deploy from branch → `main` / root → Save
4. Live at `https://<username>.github.io/proposal-platform/`
